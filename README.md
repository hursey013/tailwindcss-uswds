# U.S. Tailwind Design System

## Installation

Add the plugin to your project:

```
npm install @hursey013/tailwindcss-uswds
```

## Usage

To use the plugin, add it to your Tailwind config file.

```js
// tailwind.config.js
module.exports = {
  plugins: [require("@hursey013/tailwindcss-uswds")]
};
```
